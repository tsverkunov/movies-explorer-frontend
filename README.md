## Запуск приложения

Для запуска приложения необходимо установить [*Node JS*](https://nodejs.org)  
Клонируйте код на свою машину. 
```sh
git clone
```

Установите необходимые пакеты с помощью команды
```sh
npm install
```

Запустите приложение командой
```sh
npm start
```
